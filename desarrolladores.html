Nuestro Data Set

Para el entrenamiento del modelo de red neuronal se ha creado un data set de billetes de Guatemala que incluye billetes de las categorías: 5, 10, 20, 50 y 100 quetzales. Para cada categorías se han tomado un promedio de 190 fotos, que incluye fotos con diferentes fondos, en distintos angulos y distintas distancias. Ya que las imagénes son insuficientes para una buena red neuronal se recomienda generar imagenes a partir de las imagenes originales. Aplique cambios a las imagénes generadas como recortes, angulos, escalamientos, etc.

Obtenga el dataset desde repositorio en github. Haz click aqui para obtener el dataset.
